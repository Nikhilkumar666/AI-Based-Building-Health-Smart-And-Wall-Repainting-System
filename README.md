# 🏠 AI-Powered Building Health Analysis & Smart Wall Repainting System

An intelligent, AI-driven web application designed to analyze building conditions and generate realistic wall repainting on the same image. This system combines computer vision and deep learning techniques to provide practical solutions for building maintenance and interior design visualization.

---

## 🚀 Overview

This project focuses on solving real-world problems in construction and interior design by integrating:

* **Structural damage detection**
* **AI-based wall repainting (same image, not regenerated)**
* **Smart maintenance recommendations**

The system ensures realistic outputs by preserving original textures while intelligently modifying only the required regions.

---

## 🔍 Key Features

### 🧠 AI-Based Damage Analysis

* Detects **structural cracks** and **water leakage**
* Calculates severity levels (**Low / Medium / High**)
* Provides **AI-generated explanations**
* Suggests **maintenance actions**
* Displays **building health score**

---

### 🎨 Smart Wall Repainting (Core Feature)

* Detects wall regions automatically
* Repaints **only the wall (not the entire image)**
* Uses **AI inpainting (Stable Diffusion)**
* Preserves lighting, texture, and realism
* Generates **before vs after comparison**

## ⚙️ How It Works

1. User uploads an image of a building or wall
2. System analyzes the image using computer vision
3. Damage is detected and classified
4. Wall regions are identified using segmentation
5. AI repaints only the wall using inpainting
6. Final output is displayed with comparison and report
